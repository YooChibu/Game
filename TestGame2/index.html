게임 설명

  • 스페이스바: 다음 웨이브 시작
  • 마우스 클릭: 타워 설치/업그레이드
  • 타워 종류:
    • 기본 타워 (50골드): 기본적인 공격력과 범위
    • 스나이퍼 타워 (100골드): 관통 공격 가능
    • 스플래시 타워 (150골드): 범위 공격과 감속 효과
    • 레이저 타워 (200골드): 지속 데미지 공격
  • 타워 조합:
    • 기본 타워 + 스나이퍼 타워 = 관통 데미지 증가
    • 스플래시 타워 + 레이저 타워 = 범위 데미지 증가
    • 모든 타워 종류 설치 = 전체 타워 강화
    • 원소 지배: 모든 타워 종류 설치 + 레벨 3 이상 = 특수 효과 100% 강화
    • 전술적 배치: 인접한 타워들이 서로의 공격력 20% 증가
  • 특수 능력:
    • 전체 타워 강화: 모든 타워의 데미지 2배 (30초)
    • 적 이동 속도 감소: 모든 적의 이동 속도 50% 감소 (20초)
    • 골드 자동 획득: 5초마다 10골드 자동 획득 (60초)
  • 특수 이벤트:
    • 골드 러시: 모든 적 처치 시 골드 2배 (30초)
    • 타워 강화: 모든 타워의 공격력 50% 증가 (20초)
    • 적 약화: 모든 적의 체력 30% 감소 (25초)
  • 보스 몬스터:
    • 탱크 보스: 높은 체력과 방어 능력
    • 스피드 보스: 빠른 이동과 순간 이동 능력
    • 소환사 보스: 적 소환 능력
    • 탱크 보스: 지진 능력으로 주변 타워 무력화
    • 스피드 보스: 시간 왜곡으로 타워 공격 속도 감소
    • 소환사 보스: 타워 약화 능력 추가
  • 난이도:
    • 쉬움: 더 많은 골드와 생명력, 약한 적
    • 보통: 기본 설정
    • 어려움: 적은 골드와 생명력, 강한 적
  • 업적:
    • 첫 타워: 첫 타워 설치
    • 보스 킬러: 첫 보스 처치
    • 타워 마스터: 모든 타워 종류 설치
    • 웨이브 마스터: 10웨이브 클리어
    • 타워 전문가: 타워를 10레벨까지 업그레이드
    • 골드 수집가: 총 10000 골드 획득
    • 이벤트 마스터: 모든 특수 이벤트 경험
  • 타워 업그레이드: 설치된 타워를 클릭하여 업그레이드 가능
  • 타워 판매: 업그레이드 메뉴에서 판매 가능 (설치 가격의 70% 환불)
